🔥 Firebase — Đồng bộ đám mây ● Chưa kết nối

Kết nối Firebase để đồng bộ văn bản, đơn thư và file PDF lên cloud — mọi thiết bị đều thấy dữ liệu giống nhau theo thời gian thực. Storage Bucket là bắt buộc để đồng bộ PDF.
📄 Tự động điền từ file config — Không cần nhập tay
Tạo file firebase-config.txt chứa thông tin từ Firebase Console rồi chọn bên dưới:
Hoặc nhập thủ công:
🔗 Link chia sẻ — Mở trên thiết bị khác để tự kết nối:
✅ Đồng bộ realtime: văn bản, đơn thư, cán bộ
✅ PDF → Firebase Storage (hỗ trợ file lớn, không giới hạn)
✅ Thiết bị mới: dùng link chia sẻ để tự kết nối
📌 Cần bật Firebase Storage: Console → Storage → Get started
